Prompt Create a single file p5.js sketch that tests a drawAnimatedWire() function.

Create a function called drawAnimatedWire(x1, y1, x2, y2, speed1, state) that will display small circle moving down the wire.
The input parameters for the drawAnimatedWire() are:

  1. x1, y1: the begging point of the animated wire
  2. x2, y2: the ending point of the animated wire
  3. currentSpeed: the speed of the electrons moving down the wire
  4. state: the state of the wire (on or off)
The function should use the millis() function with a modulo to draw the process of electrons moving down the wire.
Draw four wires in a square to demonstrate the drawAnimatedWire() function.
Add a on/off button at the bottom of the canvas that turns the circuit on and off.
Add a range control slider at the bottom of the canvas to change the animation speed.
